title: Shell suite

import { LinkCard, CardGrid, Aside } from '@astrojs/starlight/components';

The **shell suite** is four small packages that work together to put a real PTY on the other end of a browser tab. You pick the transport (WebSocket for the simple case, WebRTC P2P for NAT-traversing peer connections) and the packages take care of the rest.

<Aside type="caution" title="Security model">
  Treat access tokens like SSH keys. The WebSocket server defaults to binding `127.0.0.1` for a reason. P2P offers end-to-end encryption via a shared password вҖ” see the server reference for the full threat model.
</Aside>
