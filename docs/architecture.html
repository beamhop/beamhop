Editing This Doc

it's alive 🌱

This page redraws itself from data.

Open this file and scroll to the <script> block near the bottom. All the content lives in three arrays: COMPONENTS, JOURNEYS, and GLOSSARY. Edit the data — never the markup — and everything (the map, the wires, the journeys, the cheatsheet) re-renders automatically.

Add a box? push a new entry to COMPONENTS with an x/y position (in %), a color, and a talksTo list of other ids. Add a story? push to JOURNEYS with ordered steps. New word to explain? push to GLOSSARY.

When we discuss an architectural change, Claude updates these arrays and the picture stays true to the code. No build step, no dependencies — just save and refresh.
